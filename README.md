# Mark Kempa — Healthcare Finance, AI-Native

Interactive finance tools and operating models I've built for venture-backed,
value-based-care healthcare companies. Each is a self-contained, recomputing model —
finance-first, AI-built.

## Projects
- **VBC Capitation Dashboard** — multi-state, center-driven value-based-care model
  (center rollout, panel ramp, MA contract mix, RAF & TCOC levers, capitation cash flows).
- **Cleardose FP&A** — AI-assisted FP&A dashboard + synthetic GL data engine.
- **GlowMD Aesthetics** — DTC med-spa operating & unit-economics model (+ franchise model).
- **Medicare Reimbursement Explorer** — how a Medicare physician rate is built
  (RVUs × GPCI × conversion factor), facility vs. office, by locality.
- **Digital Health Weekly Feed** — an auto-refreshing TLDR of the newsletters & shows I follow.

## Stack
